#Linux Build#
```Bash  
git clone https://tkattanek@bitbucket.org/tkattanek/oldscool_intro.git
cd oldscool_intro
mkdir linux_build
cd linux_build
cmake -DCMAKE_BUILD_TYPE=Release ..
make -j4
```
